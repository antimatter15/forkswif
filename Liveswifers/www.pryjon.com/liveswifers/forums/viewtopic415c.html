because i hated it when i had to search around the whole site for one peice of code, ive made an engine so you dont have to

put this in the enterframe
Code:
gravity++;
   _y += gravity;
if (Key.isDown(Key.LEFT)) {
        _x -= speed;
}

if (Key.isDown(Key.RIGHT)) {
        _x += speed;
}

if (this.hitTest(_root.ground)){
    _root.player.isJump = 0;
      _root.player.gravity = 0;
      _root.player._y--;
}
   if (Key.isDown(Key.UP) && isJump == false) {
      isJump = true;
      _y -= 5;
      gravity = -18;
   }


and this in the onload

Code:
speed = 4;
yspeed=0;
isjump=true;




remember to make your characters instance name "player"

and make your platform instance name "ground"
