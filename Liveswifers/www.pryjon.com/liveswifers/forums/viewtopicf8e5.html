I placed this code in the onmousemove() of the character sprite on the main stage ("Scene 1").
Code:
x1=_root._xmouse;
y1=_root._ymouse;
var obj=this.arrow;
x2=obj.pivot._x+obj._x+obj._parent._x;
y2=obj.pivot._y+obj._y+obj._parent._y;
obj._rotation=(Math.atan((y1-y2)/(x1-x2))*180/Math.PI)+(x1<x2?180:0);

Set obj to the path to the gun object, and don't forget to make a pivot point! You may want to place this code also (meaning a copy, in both places) in the place you put the code to make the character move (onkeydown, or whatever). Let me know if you need help. Ok, ok, ready for me to rock your world? Ok, first, this requires a working rotate sprite like earlier in the thread. Next, create a bullet sprite with a small ball in the top left corner, copied and pasted into three frames (yes, copy and paste, there is no need to create a whole new sprite by using transform motion.) and in the first frame, place the following code:
Code:
var speed=5; //this will need to be set properly
var xRun=0;
var yRun=0;

function initialize(angle, x, y){
   /*when you create the bullet using _root.attachMovie('bullet'),
   you will need to initialize it by calling the initialize function
   with a parameter of the gun object's rotation and _x,_y*/
   var pi=Math.PI;
   var a=angle; // degrees
   var r=pi*(a+360)/180; // radians
   var xDir=(-1);
   var yDir=(-1);
   for(i=a+90;i>0;i-=180){
      xDir*=-1;
   }
   for(i=a;i>0;i-=180){
      yDir*=-1;
   }
   var xPrep=Math.abs(1/Math.cos(r));
   var yPrep=Math.abs(1/Math.sin(r));
   
   this.xRun=xDir*Math.sqrt(speed*speed*Math.pow((yPrep/(xPrep+yPrep)),2)); //number to add to _x every refresh
   this.yRun=yDir*Math.sqrt(speed*speed*Math.pow((xPrep/(yPrep+xPrep)),2)); //number to add to _y every refresh
   this._x=x+7*xRun; // change this to a different number *xRun if necessary
   this._y=y+7*yRun; // change this too
}


Ok, second frame!
Code:
this._x+=xRun;
this._y+=yRun;
if(Math.abs(this._x)>1000||Math.abs(this._y)>1000){
   removeMovieClip(this); // make sure there are not too many sprites in the scene (helps speed up game)
}

And third frame...
Code:
this.gotoAndPlay(2);


Ok, now, you need to export the bullet sprite as asset named "bullet", and add a single instance to the scene. Finally, on the turning sprite instance already on the scene (the actual gun object), place this code in the on(mouseUp):
Code:
_root.attachMovie('bullet','bullet',bullets);
x=this._x;
y=this._y;
angle=this._rotation;
_root.bullet.initialize(angle, x, y);
_root.bullet._name="bullet"+bullets;
bullets++;


Ok, I think this tutorial is complete. Tell me if I missed anything, or if you can't get it to work.
For extra fun, make the second frame of the bullet contain a weird shape like a heart or star, or something with a semi-transparent edge. Now move the on mouse up code to on mouse move. The fun never ends!
