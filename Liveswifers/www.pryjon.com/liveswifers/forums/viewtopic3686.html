OFFICIAL CONTEST!
Whoever creates the best site design either by editing my images and CSS or by creating their own nearly fully functional site by November 30th wins.

Requirements for a "nearly fully functional site":
Site must be able to be posted to any of a large number of hosts (this means that it can use PHP, Perl, even ASP, but not ASPX or JSP).
Site must support multiple logins (usernames and passwords).
Site must have support for a downloads section, a swf gallery, and a working contact page (cannot use <form action="mailto:*">) at minimum, with the ability to add other modules easily.
Site content (not necessarily the code, but the words or gallery images of any given page) must be editable from the web without using FTP.
Site code and graphics must not infringe on any copyright/patent/trademark.
Users of this forum must be able to test the editing features using the username and password demo and demo. (This means you need to build in some security!) Not all features need to be available to that username (example, you probably don't want to let those users edit the home page content).

Eligibility:
Any person or group of persons may participate. Teams must be declared by the time of submission. All team members must be credited for the work, and a list of who did what must accompany the submission.

Submission:
All submissions must be submitted by posting as a reply to this topic, and must by submitted by 11:59pm (GMT) November 30, 2007. The post must contain the list of participants, what each participant did (a huge description is not necessary, just five words or so for each person.), a short description of tools/process used to create the site, and the URL to the working site.

Winners:
Winners will be chosen by forum members by voting in the poll topic I will create on within about 24 hours of the completion of the contest. The poll will run for 2 weeks. Participants should be judged by the forum members on the basis's following:
Required features
Extra features
Visual design and aesthetics
Ease of use and navigation
Winners will not be chosen based on content, and the entirety of the content is subject to change.


P. S. for those who wish simply use my site may rest assured that I will have implemented multi-user support and a SWF gallery by the time on the ending of the contest.
