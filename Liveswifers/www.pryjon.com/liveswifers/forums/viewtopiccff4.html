Punching code
Put sprites inside the player sprites.
Example

Player1
-Body
-Fists
Player2
-Body
-Fists

Then try this code

if(_root.Player1.Fist.hitTest(_root.Player2.Body))
{Player2HP = Player2HP - 10}

This looks for collisions with only the fist and the body and not fist on fist
