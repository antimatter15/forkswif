If
Code:
If(counter2==10);
gotoandstop(9);

is your code, you should have it like this:
Code:
if(counter2==10) {
gotoandstop(9);
}

The reason is that the semicolon ( ; ) tells actionscript that it is the end of the line of code. For the If statment, this means calculate if what is in the parenthesis is true, and that is it. Hope this helps!
