This tutorial will teach you how to make a sprite move with just a sprite and some actionscript.

Creating the sprites:
Goto>Insert>Sprite

Draw your image, but i keep it with a circle

The Actionscript:

Insert your sprite into your workspace, and select it with the select tool.

Sprite action: Onclip(enterframe)

Here comes the actionscripting to move the character arround with the arrow keys:

Code:

if (key.isDOWN(Key.UP)) {
this._y -= 4;
}   
if (key.isDOWN(Key.DOWN)) {
this._y += 4;
}
if (key.isDOWN(Key.LEFT)) {
this._x -= 4;
}
if (key.isDOWN(Key.RIGHT)) {
this._x += 4;
}


Noticed the 4? That’s the FPS for the sprite, so if you insert instead of the 4, 20 your player will walk more smoother.

If you want to learn more about making games, check the samples from Liveswif 2.2 File>Samples>Advanced>Game.lmv
