This tutorial will teach you how to make make a "sniper scope" view

1 ) Using the circle tool, draw a circle as big as the canvas, be sure to zoom out before doing so

2 ) Fill that circle with a color

3 ) Select the contour tool

4) At the bottom below the Actionscript box, there is a drop-down menu and two radio options. Select the 1st radio option right of the drop down menu

5) You will see 4 points around the circle, drag on of them outwards, you will see the outline of the view hole.
NOTE: This will be your only chance unless you go to Edit>Undo

6 ) Let go to of point when you have the desirable shape.
If you want a target marker, just keep dragging a point outwards until you have enough rings

If you wish to use it as a custom cursor, you may proceed

7 ) Select the shape and cut it, go to Insert>Sprite

8 ) Follow the Custom Cursor/Invisible Mouse Pointer tutorial, the shape will be your custom cursor

Invisible Mouse Pointer:
To make the mouse pointer invisible, type in the code
Code:

mouse.hide();


on the frame to make the mouse pointer invisible, from then on, the mouse pointer will remain invisible until you use the code
Code:

mouse.show();

, and from that frame on, the mouse pointer will remain visible
------

Custom Cursor:
1 ) Create a sprite with the cursor shape. The mouse point is focused on the upper left corner, adjust the shapes position to match with the mouse point coordinates

2 ) Go back to the canvas and drag the sprite onto the canvas.
[If you stop here proceed to step 3a]

If you wish for the custom mouse to last for a period of frames, you may add a transform motion to that sprite
[If you use this set of extra instruections proceed to step 3b]

3a ) Add the code
Code:


this._x = _root._xmouse;
this._y = _root._ymouse;


onto the onClipEvent(enterFrame)

3b ) Go to both of the frames where the transform motion begins and ends . There, add the code

Code:

this._x = _root._xmouse;
this._y = _root._ymouse;


onto the onClipEvent(enterFrame)
for both of the frames containing the sprite
