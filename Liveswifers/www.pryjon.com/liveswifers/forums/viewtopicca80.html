Yeah, you can do the docs. Play around with it to see how to do things, and report all the bugs you find.

the data will be in OPF format (for now). which is JSON. But for starters, use JSON to decode a single shape in the format of

{
"id":"shape:f9aca501-015e-ca70-5c24-e487ea703ebf",
"type":"rect",
"transform":"rotate(0, 65, 85)",
"fillColor":"#FF0000",
"lineColor":"#000000",
"fillOpac":1,
"lineOpac":1,
"lineWidth":1,
"left":37,
"top":58,
"width":56,
"height":54
}

You can decode it with php's json_decode() and put them into freemovie. If you need help, please ask.
