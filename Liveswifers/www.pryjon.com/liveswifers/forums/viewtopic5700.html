Suggested knowledge:

  • Locations of files ( Such as C: )
  • Novice Experience with computers


With this, you can put your Liveswif creations on your desktop.

First, create your .swf with anything in it, such as a game or clock. It is suggested to make a .swf file lager than your screen size.

Then, save it anywhere, we're going to name it background.swf.

Next, open up a text editor, such as Notepad. We are going to be using a html document for this.

Copy and paste this code onto the document:
Code:

<object>
<param name="movie" value="file:///x/background.swf">
<embed src="file:///c:/WINDOWS/background.swf" quality="high" type="application/x-shockwave-flash" width="w" height="h"></embed>
</object>


Replacing
"x" with the location of background.swf
"h" with the height of background.swf
"w" with the width of background.swf

Save the file as background.html, in the "All files" category.

Then go to "My Computer" and change your desktop background.

Select the file of background.html, and set that as your background.
