Well, here is the password tutorial in text format:

First, open liveswif and set the movie properties: Width=300, Height=200,
frame rate=20, and add a nice background color

Next, We will draw the next things:
Background
A bar
The letters
A button
A rectangle

Background:
Put the word: "Password:" in the first frame. and set the position to: x=40, y=92.
Draw a white rectangle set the x=144, y=86, w=90,h=29, line=0.0
That is all for the background.

The bar:
Select new sprite and draw a little bar and select transform motion. Set the color motion to fade to white 100%. Drag it to the first frame
In the OnClipEvent[enterFrame] put the next code

Code:
if(this._x>75){
this._x=75;}
if(this._x<1){
this._x=0;}


In the onClipEvent[keyDown]:
Code:
recon=Key.getAscii();
if(recon==8){
   this._x-=15;}
if((recon>96)&&(recon<123)){
   this._x+=15;}

In the onClipEvent[load]:
Code:
this._x=0;


The letters:
Select new sprite, in the first frame draw a "1" put the next code in the on frame action:
Code:
stop ();

Go to the frame 97, draw an "a", go to next frame (9 and draw a "b", etc. Do this until the frame 122 who represents the letter "z".
Drag it into the main frame and give the instance name="letra1"
Drag other letter animation into the first frame and name it "letra2"
Do this until "letra5", because the password allows 5 letters

The rectangle:
The rectangle is the most important thing in the password. Select new sprite, draw a small rectangle and drag it into the main frame
in the onClipEvent[enterFrame]:
Code:
if((reco>96)&&(reco<123)){
 if(_root.control==1){
    _root.letra1.gotoAndStop(reco);
   _root.cont1=reco;}
 if(_root.control==2){
    _root.letra2.gotoAndStop(reco);
   _root.cont2=reco;}
 if(_root.control==3){
    _root.letra3.gotoAndStop(reco);
   _root.cont3=reco;}
 if(_root.control==4){
    _root.letra4.gotoAndStop(reco);
   _root.cont4=reco;}
 if(_root.control==5){
    _root.letra5.gotoAndStop(reco);
   _root.cont5=reco;}
  if(_root.control>5){
     _root.control=6;}}
if(reco==8){   
if(_root.control<5){
_root.control+=1;}
if(_root.control==5){
    _root.letra5.gotoAndStop(1);}
 if(_root.control==4){
    _root.letra4.gotoAndStop(1);}
 if(_root.control==3){
    _root.letra3.gotoAndStop(1);}
 if(_root.control==2){
    _root.letra2.gotoAndStop(1);}
 if(_root.control==1){
    _root.letra1.gotoAndStop(1);
   _root.control-=1;}
_root.control-=1;
if(_root.control<0){
   _root.control=0;}} 


In the onClipEvent[keyDown]:
Code:
reco=Key.getAscii();
if((reco>96)&&(reco<123)){
   _root.control+=1;}
if(reco==8){
   _root.control-=1;
   _root.cont5=0;}


in the onClipEvent[load]:
Code:
_visible=false;


The button:
Draw a button and drag to the main frame
Put the next code in the OnRealease action:
Code:
if((_root.cont1==101)&&(_root.cont2==110)&&(_root.cont3==116)&&(_root.cont4==101)&&(_root.cont5==114)){
_root.gotoAndPlay(2);}


The last thing:
Put the next code in the onFrame action of the main frame:
Code:
stop();
control=0;
cont1=0;
cont2=0;
cont3=0;
cont4=0;
cont5=0;


In the second frame draw something and put the next code in the onFrame action :

Code:
stop();


That is all.
