Did you right-click the sound file in the right-hand side of the screen, click "Export as Assets" and gave it a name like in this tutorial?

If so, say you named the sound Music. The code to start the sound would be

Code:
yourSound=new Sound(this);
yourSound.attachSound("Music");
yourSound.start(0, 99);


... And the code to end it would be

Code:
yourSound.stop("Music");
