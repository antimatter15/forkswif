I implemented the Manual and FAQ. Follow these instructions.

First, insert this in the empty <script></script> section in the HEAD of the index.html:
Code:
function window_in(attr)
{
document.getElementById(attr).style.left="25"
}
function window_out(attr)
{
document.getElementById(attr).style.left="-860"
}


Right below that, insert this in the empty <style></style> section:
Code:
.brwainer {
position:absolute;
left:-860;
top:25;
width:750px;
height:560px;
z-index:2000005;
background: #FFFFFF;
}


Next, insert this right below the <body> tag:
Code:
<div id='manual' class='brwainer'><a href="#" onclick="window_out('manual')">X</a><iframe height="550" width="750" src="http://www.freewebs.com/ajaxanimator/manual.htm"></iframe></div>
<div id='faq' class='brwainer'><a href="#" onclick="window_out('faq')">X</a><iframe height="550" width="750" src="http://www.freewebs.com/ajaxanimator/faq.htm"></iframe></div>


Finally, insert this in the menu portion of the html, right after ...Timeline...<li>...</li>:
Code:

   <li id="55003"><a href="#">Help</a>
            <ul width="150">
               <li id="550031" jsFunction="window_in('manual');"><a href="#">Manual</a></li>
               <li id="550032" jsFunction="window_in('faq');"><a href="#">FAQ</a></li>
                 </ul>
   </li>

function setOpacity(value) {

testObj.style.filter = 'alpha(opacity=' + value*10 + ')';
}
