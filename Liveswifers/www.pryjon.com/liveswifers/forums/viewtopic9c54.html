How to play animation when pressing down on key

first open up a new sprite

now get a character or make one

make a walking animation No transform motion!!!

now pretend your code for move right is this(below) and your walking animation starts on frame 2 of the sprite
Code:
if (Key.isDown(Key.RIGHT) == true) {
      x = hforce;
   }


so you would add
Code:
gotoandplay(2);

like this
Code:
if (Key.isDown(Key.RIGHT) == true) {
      x = hforce; gotoandplay(2);
   }


so lets say your animation for walking is 5 frames (so that means frames 2,3,4,5,6

so on your characters standing frame (probably frame 1) you have the code
Code:
if (Key.isDown(Key.RIGHT) == true) {
      x = hforce; gotoandplay(2);
   }

on frame 2 change gotoandplay(2); into gotoandplay(3);

on frame 3 change gotoandplay(3); into gotoandplay(4);

repeat until on last frame of walking animation then put gotoandplay(2); (so it could repeat)

Create a sprite
In frame 1, put this code in the onframe thing:
Code:
if (Key.isDown(Key.{insert Key here}) == true) {
      gotoandplay(3);
   }

[Yes, this is an edit of the code above. Cry me a river]
In frame 2, put
Code:
gotoandplay(1);

In frame 3, begin the animation. The same rule applies as the first post, no transform motions.

If it's something like oh say, a sword being swung only once, your job is done, but if it's something more complex like a walk cycle

==Note==
This is if you're doing something like a walk cycle, not a simple weapon attack
Also, I don't know if this will actually work, I haven't done this yet
==Note==

put this
Code:
gotoandplay(1);

At the end of each action and alter the code. For example, if you wanted a sprite that turns when it goes left or right, I imagine you would have to put in this code in the onFrame {whatever you call it, I forgot}:

{frame 1}
Code:
   
if (Key.isDown(Key.right) == true) {
     gotoandplay(3);
   }
if (Key.isDown(Key.left) == true) {
     gotoandplay(frame the walk cycle begins on);
   }

this code in frame 2 and at the end of each walk cycle:
Code:
gotoandplay(1);
