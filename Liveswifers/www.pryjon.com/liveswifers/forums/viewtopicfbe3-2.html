An example here:
http://www.pryjon.com/star.html

Download the LMV file:
http://www.pryjon.com/stars.lmv

Step 1 -

In movie properties change your background to Black and set your dimensions to 600 x 300

Create a sprite that looks like a star. A round white circle will do.

Return to Scene 1 and right click the star.

Click "Export as assets" and give it the name "dot".

In frame 1 in the "On Frame" event put in this code:


Code:
this.count = 120;
this.speed = 1;
this.focaldistance = 25;
this.rmin = 50;
this.rmax = 1000;
this.zmin = 10;
this.zmax = 300;
var i = 0;
while (i < this.count) {
   this.attachMovie("dot", i, i);
   ++i;
} // end while
   
function placeInSpace(id) {
      var ratio = this.focaldistance / Math.sqrt(this[id].z * this[id].z);
      this[id]._x = (this[id].x * ratio) + 300;
      this[id]._y = (this[id].y * ratio) + 150;
      this[id]._xscale = this[id]._yscale = 1000 / this[id].z;
   };
function randomPlace(id) {
      var radius = this.rmin + (this.rmax - this.rmin) * Math.random();
      var polar = 2 * Math.PI * Math.random();
      this[id].x = (radius * Math.cos(polar));
      this[id].y = (radius * Math.sin(polar));
      this[id].z = this.zmin + (this.zmax - this.zmin) * Math.random();
   };


In frame 2 enter this code:

Code:
var i = 0;
while (i < this.count) {
      if ((this[i].z = this[i].z - this.speed) <= 0) {
         this.randomPlace(i);
         }
   this.placeInSpace(i);
   i=i+1;
}


And in frame 3 enter this code:

Code:
gotoAndPlay(2);


You can play with some of the variables in frame 1 to get different effects and change speeds.

The center point of the stars is determined in the placeInSpace() function. The +300 and the +150 are the offsets for the dimensions of the movie. If you were to make this 300x300 for instance you would want to change those 2 numbers to 150x150 (half of the original).

Anyways, thought this would be fun!

I got the code from another site and changed it around a bit to work in Liveswif. If you want to know more about how it all works go here:

http://jrgraphix.net/research/flash_starfield.php

But don't put five starfields (one in each corner and in the center) in the same frame. I did for a flash-made website I tried making, but it caused the file to not load anymore. So the general rule of thumb for this starfield is:
Not more than one starfield per frame (main timeline or otherwise)
