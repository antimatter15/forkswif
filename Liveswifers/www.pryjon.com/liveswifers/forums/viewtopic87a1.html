Onion skinning is when you are able to see the previous frame while animating. This allows smoother animation since the user does not have to "guess" the position and shape of the object being animated in the last frame.

It's rather similar to having a transparent sheet over the previous frame and using it as a guide, like with some animation studios, while editing a frame.

Many Flash animation programs, even Flash itself, have this feature.
