To make an object disappear via ActionScript, use the _alpha property like so:
Code:
function fade(){
  for(var i=0;i<100;i++){
    this._alpha=100;
  }
}


Now, for clicking multiple times, put into the victim sprite a variable called shot_count. Initialize it to zero, and ever time they are shot, increase it by one. Once it has reached 3 (or whatever) an "if()" statement should be able to take care of beginning the dieing animation.
