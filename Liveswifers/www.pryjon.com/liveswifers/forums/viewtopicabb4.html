Transform Motion

With this mode of animation we can tell an object or sprite to move from one place to another, to scale from one size to another, or to change color.

Firstly, in frame 1 set your object where you want it to start.

After that, select your object and click INSERT>Transform Motion

A blue line will appear in our TIMELINE. This line indicates the FROM and TO positions of the object and how many frames it takes to get there.

We can adjust the number of frames or the start frame by simply clicking the blue dots and moving them forward and back on the TIMELINE.

As you can see, Liveswif has automatically chosen 36 frames for our object's Transform motion.

Select frame 36 on the TIMELINE. Now we can move, or scale the object by simply changing the properties of it.

So go ahead, move the object to another position on the screen or change its scale. Press the PLAY button and watch it happen!

You can also use the Transform motion tool to change the color, or alpha of an object. This function is located to the right of the color pallete. We can choose to change the color or alpha by clicking the option and adjusting the settings.

Path motion

Path motion is a very useful Liveswif tool. An object can circle an object, or can perfectly follow a designated path.

Firstly, create an object. A square does nicely because it will give us some idea of which way it is facing.

Now use the spline or curve tool to create a path. Any which way will do.

In order for the program to know which object is the path and which is the follower, we need to select them in order.

First, select the path. Then (by holding down the SHIFT key) select the object.

Now that we have 2 objects selected click INSERT>Path motion

The object should have connected to the begining of the path and our TIMELINE should have a 36 frame blue line (like in Transform motion.)

We can adjust the scale or color of the object while it is in this mode just as we could in Transform motion.

Press PLAY and the object should move along the path from begining to end in exactly 36 frames.

The object also faces in the direction of the path. As of yet, there is no way to avoid this. Correct me if I am wrong.

Keyframe animation

Keyframe animation is about the most straight forward and simple way to animate in Liveswif. Each frame on the TIMELINE would have to be edited seperatly.

First, create a box.

Then move to frame 2.

Create a new box in a different location.

Press PLAY.

Very simple, but can be very useful too!
