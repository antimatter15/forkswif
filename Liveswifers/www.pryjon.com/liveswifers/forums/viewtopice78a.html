Alpha Transparent
Finally, 2 good things from this. The first thing is you can say good bye to that white box around your imported image, and second is you will be able to import 3D modles.

Why?
Because LiveSwif can support TGA Files!

Here is how:
1) Download TrueSpace 3: http://www.caligari.com/products/truespace/ts3/default.asp. NOTE: YOU NEED TO USE YOUR REAL EMAIL BECAUSE THEY SEND YOU THE REGITRATION KEY
2) Install TrueSpace 3
3) Make a image to import, let's just make something like TEST.bmp. Make sure the background is white
4) In TrueSpace 3, go to: EDIT -> Image Utilities
5) under "Input Image" Seclect the image you want to import into LiveSwif.
6) Set Alpha Conversion to "White"
7) Click "Convert Image"
Cool Make sure you save it as a TGA File
9) You can also make 3D TGA files to use in LiveSwif

An example here: http://img383.imageshack.us/my.php?image=animatedlinkzm7.swf

Now let’s get started!

If i’m right, you see how to convert your sprites above.

Open liveswif 2.2
Goto>Insert>Sprite
Goto>Insert>Image

Import all your sprites, select them all. And cut them all. Except the first sprite you want to use

Select your image
Align to canvas center

Now you imported a image on the first frame. Go to frame 2

Paste back your sprites.
Use the second sprite

And you go further like that.
Until you have something like this:

Go to the last frame of the motion.
And move your animation to the right/left/up/down

Transform Motion

With this mode of animation we can tell an object or sprite to move from one place to another, to scale from one size to another, or to change color.

Firstly, in frame 1 set your object where you want it to start.

After that, select your object and click INSERT>Transform Motion

A blue line will appear in our TIMELINE. This line indicates the FROM and TO positions of the object and how many frames it takes to get there.

We can adjust the number of frames or the start frame by simply clicking the blue dots and moving them forward and back on the TIMELINE.

As you can see, Liveswif has automatically chosen 36 frames for our object's Transform motion.

Select frame 36 on the TIMELINE. Now we can move, or scale the object by simply changing the properties of it.

So go ahead, move the object to another position on the screen or change its scale. Press the PLAY button and watch it happen!

You can also use the Transform motion tool to change the color, or alpha of an object. This function is located to the right of the color pallete. We can choose to change the color or alpha by clicking the option and adjusting the settings.
