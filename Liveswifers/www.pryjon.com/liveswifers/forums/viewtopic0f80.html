Welcome to Liveswifers!

Here is what I would do:
1. Make a sprite. In frame 1 of the sprite put the code stop(); in the onFrame event.
2. In frame 1 of the sprite, put a button on the frame that has code something like:
Code:
sound.start();
nextFrame();

3. In frame 2 of the sprite, put a button on the frame with code something like:
Code:
sound.stop();
prevFrame();


That is how I would do it.
