Preamble
-------------------------------------------------------------------

This new Liveswifers.org Law has been created so that both the user and the moderator/administrators both know what the rules are. These laws are put up so the user knows what will get them punished. These laws are put up for the mods and admins so that there is a set ruling for punishment, instead of the old way, which was based on the individual moderator's/administrator's beliefs.

Please check back frequently for minor updates. Ignorance is not an excuse for breaking the rules.

-------------------------------------------------------------------

Table of Contents:
Section i: Law Violation
Section 1: Basic Account Conditions
Section 2: Topics and Posts
Section 3: User Conduct
Section 4: Flash Licensing

--------------------------------------------------------------------

- Section i: Law Violation

This section lists how to handle times when the laws are broken

- Law i.1: Offense Counts
-- Law i.1.1: Consequences
Each user registered will be given ten Offense Counts.
Once a user reaches:

1 Offense Count: The user is notified they have received their first offense count. They are also put on the probation list.
2-4 Offense Counts: The user will be notified, at each offense, of how many counts they have left until a temporary ban.
5 Offense Counts: The user will have a temporary ban, decided by administrators.
6-9 Offense Counts: The user will be notified of how many offense counts they have left until account deletion.
10 Offense Counts: The user will have their account deleted.

-- Law i.1.2: Offense Counts Redemption
Certain users may receive a ticket that allows them to redeem a specified amount of offense counts at any time. The user may earn these by winning contests or going out of their way on the board. The tickets can only be awarded by moderators and administrators, and these same people cannot award tickets to themselves. It is possible to receive multiple tickets.
If you receive a ticket, you will be PM'd with a message showing you the details of the one you just received and (if applicable) showing the details of other tickets you have left.
Lastly: DO NOT WORRY ABOUT KEEPING TRACK. We will have a special topic open for this in the mod & admin only forum.

-- Law i.1.3: Offense Counts Total
Offense Counts shall accumulate. If you were to get 2 offense counts in spelling and grammar and 3 in offensive language, you still get a temporary ban.

-- Law i.1.4: Offense Counts Key
The following key is used to list offense counts for violation of the following laws.
xW = Warnings. x shows how many warnings you get before offense counts pile up from that law.
x(Cy) = Offense counts. x shows how many times y is added to offense counts total. y shows how much is added to offense counts total. For example, 3(C1) shows that you add 1 offense count 3 times before moving on to the next offense count (which might be 2(C2))

--------------------------------------------------------------------

- Section 1: Basic Account Conditions

This section describes certain requirements the user must perform to keep their account.

- Law 1.1: Activity
A new account must make a post within the first 7 days of account creation or the account will be deleted.
Offense Counts Chart: 1(C10)

--------------------------------------------------------------------

- Section 2: Topics and Posts

- Law 2.1: Unnecessary Topics and Posts
A user that creates an unnecessary topic/post that fits into any category of:
Already asked question by user.
Recently asked question by others.
Unrelated topics/posts.
Will receive the below Offense Counts.
Offense Counts chart: 3W, 3(C1), 2(C2), 1(C3)

- Law 2.2: Searching before asking
Users must put in some effort to solve their questions on their own, using any search tools necessary. Violation of this rule will result in receiving the below Offense Counts.
Offense Counts chart: 3W, 1(C2), 1(C3), 1(C5)

- Law 2.3: Spelling and Grammar
A user that uses incorrect spelling, grammar, or English (that has proof he/she is a fluent English speaker) will be given the below Offense Counts.
Offense Counts chart: 3W, 3(C1), 2(C2), 1(C3)

- Law 2.4: Pornography and non-related Advertisements
A user that posts pornography and non-related advertisements will be given the below Offense Counts.
Offense Counts chart: 1(C10) (automatic account deletion)

- Law 2.5: Unnecessary Polls
A user that creates a poll that is unnecessary will receive the following Offense Counts:
Offense Counts chart: 3W, 3(C1), 2(C2), 1(C3)

--------------------------------------------------------------------

Section 3: User Conduct

- Law 3.1: Language (Foul Language)
A user that uses curses or general slurs/insults will be given the below Offense Counts.
Offense Counts chart: 1W, 3(C1), 2(C2), 1(C3)

- Law 3.2: Language (Racism/Sexism)
A user that uses racial or sexual slurs/insults will be given the below Offense Counts.
Offense Counts chart: 3(C1), 2(C2), 1(C3)

- Law 3.3: Authority and Impersonation
A user the impersonates themselves as a higher authority then other users will be given the below Offense Counts.
Offense Counts chart: 3W, 4(C1), 3(C2)

- Law 3.4: Private Messages
You are expected to follow all the laws in private messages (PMs). If you have received a PM that violates these rules, please PM the exact message, word for word, to an admin or mod. We will deal out one extra offense count per bad message (not per violation), because we are unable to catch this on our own and the sender obviously thought he could get away.
Offense Counts chart: 10(C1)

-Law 3.5: Disrespect
Users are expected to respect any and all authority. Disrespecting administrators will receive more offense counts than disrespecting moderators, but nonetheless, your offense counts will shoot up. The chart is as follows:
Offense Counts chart (disrespecting moderators): 3(C2), 1(C4)
Offense Counts chart (disrespecting administrators): 1(C3), 1(C7)

--------------------------------------------------------------------

Section 4: Flash Creation

- Law 4.1: Default License
The default license for all flash, new and old, is the Creative Commons Attribution-Noncommercial-Share Alike 3.0 Unported License. The author of the flash is, of course, allowed to replace the default with any license they choose. The author must clearly note the change in licensing, though.

- Law 4.2: Plagiarism
A user that commits plagiarism with sufficient proof will be given the below Offense Counts. (Unless the owner of the original work has given permission to share his/her work)
Offense Counts chart: 5(C2)

Closing: By joining LiveSwifers you must agree to obey these rules.
