ok, so I realize my attempts at a tutorial and faq were rather futile, but i think I can help with the coding now (I've learned a lot about php recently). For the animation browser, are all the compiled flashes stored in the same folder? do they all have a .swf extension? if yes to both, then the following code will return a list of all the flashes in a directory
Code:
<?php
   $handle = opendir('INSERT LOCATION OF SWF FOLDER BETWEEN THESE QUOTES');
   while (false !== ($file = readdir($handle))) {
      if (strpos($file, '.swf',1) ) {
         $name1 = str_replace(".swf", "", $file );
         ?><p><a href="INSERT LOCATION OF SWF FOLDER HERE/<?php echo $file ?>" ><?php echo $name1 ?></a></p><?php
      }
   }
?>


For Example, this code loads all the pdf's from the 'resources/forms' directory on the website for my youth group:
Code:
<?php
   $handle = opendir('resources/forms/');
   while (false !== ($file = readdir($handle))) {
      if (strpos($file, '.pdf',1) ) {
         $name1 = str_replace(".pdf", "", $file );
         $name2 = str_replace("+", " ", $name1 );
         ?><p><a href="http://www.sgyg.org/resources/forms/<?php echo $file ?>" target="_blank"><?php echo $name2 ?></a></p><?php
      }
   }
?>


Note that the first 'path to swf folder' is relative, and the other is absolute. both MUST have a trailing "/" (backslash, i think).

P.S. if you look at my code for my youth group, you will see the I make php replace all + (pluses) with " " (spaces). this is so if I want a sapce to appear in a link I put a + in the file name at that place. This code has been tested on 110mb.com and 1and1.com (the latter is a paid host, highly recommended).
