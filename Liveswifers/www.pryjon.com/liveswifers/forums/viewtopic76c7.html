Ajax Animator is a project to create a fully standards-based, online, collaborative, web-based animation suite. Really, its goals are completely different from projects like Adobe Flash, which are not standards-based (Flash = Properietary), they're not online (~2gb of bloatednessity), aren't really collaborative, and not web-based (though Flash Player is..).

Though its not designed *for* flash, it _supports_ flash. So you can export to Flash markup, as well as to the Ajax Animator format (JSON-based), Processing (a language), Microsoft Silverlight (XAML) (AKA. eewww), and Animated GIF (OMG raster).

Ajax Animator 0.2+ Features

* Context menu on timeline
* Vector Graphic Editor (RichDraw/OnlyPaths)
* Multi-Layer Support
* Saving/Opening JSON-Based file format
* Animation Browser
* Preview
* Color Picker
* FAQ
* Manual
* Timeline-Tooltips
* ExtJS 2.1 UI
* Interactive Tutorials
* About Ajax Animator
* Tip Of The Day
* History
* Shared Clip-Art Library
* Full Undo
* Properties Panel
* Much More
