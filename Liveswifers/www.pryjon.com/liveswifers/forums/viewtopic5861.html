Added: Revision System

Basically it allows you to view (not edit) previous revisions. Really just a simple addition because of another feature.

Usage:

got to the Advanced properties tab:

Enter:
Code:
PreviewRevision(INSERT REVISION NUMBER HERE)


*replace "INSERT REVISION NUMBER HERE" with actual revision number
*revisions start at 1
*when previewing normally, (at latest revision), the max revision number will show in the status
*if input revision is larger/smaller than total, it will show a blank screen
