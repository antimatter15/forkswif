I supposes if you want to be technical, there is no plain 'Flash 9'. There is a Flash Player 9, which is a completely different beast from Flash Players 1-8 altogether. The people at Adobe created an entirely different virtual machine for .swf 's that use AS3 than those that use 1 or 2. So, if OpenSwif is modeled after Flash player 9 then it will only include AS3. If it's modeled after Adobe Flash CS3, then it'll probably have both AS3 and AS2. There is more or less no chance of it using AS1. Action Script 2 was more or less an upgrade to AS1. AS3 is almost a new language entirely.[/i]
