I didn't use _droptarget, I used hitTest. Here is how:

Preperation
In the frame with the dragging action, put this code (in the onFrame event):
Code:
stop();


Create two sprites, one the goal and one the item to be put in the goal.

The Work

Put the two sprites onto the frame. Give the goal the instance name "goal" (without the quotes). To find the instance name field, select the sprite, then at the bottom you should see a textbox saying <Instance Name> in light gray.

Give the item the instance name "item" (without the quotes)

In the item's onMouseDown event, put in this code:
Code:
if(this.hitTest(_root._xmouse, _root._ymouse, true)) {
this.startDrag();
}


In the item's onMouseUp event, put in this code:
Code:
this.stopDrag();


Now, in the goal's onMouseUp event, put in this code:
Code:
if(this.hitTest(_root.item)) {
_root.nextFrame();
}


Final Steps

Put something on the frame right after the one with the dragging.
ENJOY!

For reset, just make a button, then on the onRelease part of the button, put this:
Code:
gotoAndPlay(frame number of game);


That should work. If it doesn't, let me know.

For the eye thing, make two buttons, one left arrow and one right. On the right arrow, just put this on the onRelease:

Code:
_root.instance name of eyes.nextframe();


And this on the onRelease of the left arrow:

Code:
_root.instance name of eyes.previousframe();


Now edit the eye's sprite. Put different eye colors on different frames of it. Put stop(); on the first frame of the sprite.
