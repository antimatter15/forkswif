I don't want to code the whole thing in java for numberous reasons:
a. I pretty much suck at java
b. There is a lack of JavaScript Web Based Applications
c. Coding parts in Javascript/HTML and some server-side saving will allow you to work on the animation in a Java non-supported or not installed platform and compiling it later with a server-side service or waiting to comtinue the work on a java-enabled platform.
Actionscript would probably be the easiest to implement, as there are many many many many many many many many many many many many many many open source action-script compilers-most are easy to use with other flash tools.
SVG will only be used as a platform for this application. It basically will use some svg drawing system, and replaces the svg source generator with a ( i'm planning for it to be xml like) script the generates flash file.
Bezier curves are probably supported in SVG but it still isn't gonna be a feature in version 0.5, and I probably will quit the project for others to continue after version 1.0

I would really like it if someone else helped me, but i have a feeling nobody would even start unless I create a platform for them to begin with a near-fully functional system.
