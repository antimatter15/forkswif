I don't know if you want it to count down automatically or not, so here's the way if you don't want it to count down automatically.

I'm assuming you've made a counter. If not, insert a sprite, and on the first frame, put a three (or the highest number you want it to count down from. For the purpose of this answer, I'll use three). Then, on the next frame, put a 2, then a 1 on the third frame. Put stop(); on the first frame of the sprite. Insert a single instance of that into your main canvas. Give it an instance name of countdown . If you use something else, just replace countdown in the following code with that.

Now that you have the countdown sprite on your canvas, add an instance of a button. Then, on a button's onPress or onRelease event, put this code:
Code:
_root.countdown.nextframe();

if(_root.countdown._currentframe == 4) {
     _root.gotoAndPlay(the frame number you want it to jump to);
}


Now try it. make sure you replace the frame number that you want it to jump to. If you counted down from 10, replace the 4 with 11. That's all.

I do this all the time. To have a sprite play on a frame and then go to the next frame of the main canvas, just:

  1. Put stop(); in the onEnterFrame event of frame with the sprite.
  2. Get the sprite on said frame (if it isn't already).
  3. In the sprite's last frame, put in this code (onEnterFrame):
    Code:
    _root.nextFrame();
