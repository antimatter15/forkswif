The thing is, we don't have the source code for Liveswif, so if we were to make a Liveswif 2.3, it would have to be from scratch (and since we wouldn't have the existing Liveswif framework, this would mean no backwards compatibility!).

In fact, I remember when I first joined Liveswifers ShaneP was trying to get in touch with the creators of Liveswif to obtain the source code. This failed, so the forum's attention was turned to the development of Openswif instead.

Decompiling the program was (and still is) a possible option, but unless we know what programming language Liveswif was written with, it would be almost impossible to get any sort of useable code from an average decompiler as different decompilers are designed for different languages.

Short answer: Liveswif 2.3 is very unlikely as far as I know. Sorry!
