ok the easiest way would be to make 2 sprites with animations of the bars getting shorter. and to make them lose health it would be
Code:
armorbar.nextFrame();

and you would do this as much as you need health lost. and for the check on the armor bar i would have all dmg go to armor bar. on the armor bar add another frame past the last one with the same image as the 0 one with this code.

Code:
healthbar.nextFrame();
gotoAndPlay(x);

x is the number of the zero frame.

now you could do it a diff way with checks built into damage but this way you dont have to write extra code, its built into the sprite.
