i was able to create a new circle for each mouse click by placing this code on the onClipEvent(mouseDown) of a sprite that is on my frame.

var newCircle = "circle"+_root.count;
_root.Circle.duplicateMovieClip(newCircle, _root.count);
_root[newCircle]._x = _root._xmouse - 83;
_root[newCircle]._y = _root._ymouse - 60;
_root.count += 1;

First i created a circle sprite and gave it an instance name of Circle.

and each time i click the mouse i get a new circle where i clicked
