shanep wrote:
GTK is out. Because it requires that you install it seperately from the program.

Huh? There's plenty of apps out there that install GTK as part of the main program's installer, like Inkscape and Gaim.

Speaking of Inkscape, you might consider having that be the main tool for actually 'editing' your vector graphics. That way you could focus more on the animation & scripting side of things.

If you're looking for a 2D rendering library to build on, you might take a look at Cairo. Mozilla uses it for SVG rendering in Firefox and such, and is moving toward rendering the whole app with it. It supports several different rendering outputs, including native X, Windows, and OpenGL.
