The bar:
Select new sprite and draw a little bar and select transform motion. Set the color motion to fade to white 100%. Drag it to the first frame

Code:
 
if(this._x>75){
this._x=75;}
if(this._x<1){
this._x=0;}

Code:
 
recon=Key.getAscii();

if((recon>96)&&(recon<123)){
   this._x+=15;}

In the onClipEvent[load]:
Code:

this._x=0;

Code:

Drag it into the main frame and give the instance name="letra1"
Drag other letter animation into the first frame and name it "letra2"
Do this until "letra5", because the password allows 5 letters

Quote:

Okay, i was investigating with the "password" tutorial, and if was making it so it had different passwords. it didn't work at first, but finally I had password + choice of where he/she wanted to go

The part that i want to show you how to do, is moderate simple!

First, start out with a new document.
place a custom background.

~~~~***from here on end, it is about shapes & code.

1. create a new sprite; name the instance later on"goo".
2. create a button w/ goto: on the top.

On the new sprite, create a box in the first about this size:
._______
|_______|
figure 1
Place Goto: on the top

On the sprite goto the second frame, and copy the small box from the first frame.

create a box above it that is slightly larger.

the box should look like this: (finished)
_______
|goto:' |
|frame3 |
|frame4 |
|frame5 |
|______'|
figure 2

put buttons on each of the names (invisible)

Coding for the frame3 Button: (under "On Release")
Code:
 if((_root.cont1==101)&&(_root.cont2==110)&&(_root.cont3==116)&&(_root.cont4==101)&&(_root.cont5==114)&&(_root.goo.f==3))
 _root.gotoAndStop(3);

Coding for the frame4 Button: (under "On Release")

Code:
 if((_root.cont1==101)&&(_root.cont2==110)&&(_root.cont3==116)&&(_root.cont4==101)&&(_root.cont5==114)&&(_root.goo.f==4)
 _root.gotoAndStop(4);


Coding for the frame5 Button: (under "On Release")
Code:

if((_root.cont1==101)&&(_root.cont2==110)&&(_root.cont3==116)&&(_root.cont4==101)&&(_root.cont5==114)&&(_root.goo.f==5)
 _root.gotoAndStop(5);
[color=yellow]

goto frames 3-5, and copy the button from figure 1

over the button on frames 3-5, put the following words over the button:

on frame 3 put:
"frame3"
on frame 4 put:
"frame4"
on frame 5 put:
"frame5"

YOU HAVE JUST MADE THE BEGINNING OF THE COMBO BOX!

coding for the three frames on the "combo box"

goto frame 3, and put on the frame coding:
Code:

stop();
f=3;



goto frame 4, and put on the frame coding:

Code:

stop();
f=4;



goto frame 5, and put on the frame coding:


Code:

stop();
f=5;
