first create your ball with the instance name "ball"

then create a platform with the instance name "ground"

put this code the balls enterframe

Code:
if (this.hitTest(_root.ground)) {
   yspeed=-4;
}

and this in the onload


power = 0.7;
yspeed = 0;
xspeed = 0;
friction = 1;
thrust = 0.75;
accelSpeed=0;
isJump = true;
gravity = 1;



and to make it move correctly put this in the enterframe

Code:
if (Key.isDown(Key.LEFT)) {

        xspeed -= power;

}

if (Key.isDown(Key.RIGHT)) {
   
        xspeed += power;
}
if (Key.isDown(Key.UP)) {

        yspeed -= power*thrust;
}

if (Key.isDown(Key.DOWN)) {

        yspeed += power*thrust;
}
   
   xspeed *= friction;
    yspeed += gravity;
   
    _y += yspeed;
    _x += xspeed;
 


thank you dezireth for your advanced movement code, and all the people from pspdemocenter who gave me the hitTest codes (mostly cay311)
