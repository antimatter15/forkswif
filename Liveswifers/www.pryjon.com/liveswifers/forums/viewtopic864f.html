Microsoft SilverLight. is a relatively new and different version of flash.


Features:
1. It is free.
2. Design is XML based
3. Scripting is Javascript based
4. Media!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!! (Videos, and Music)
5. High-Speed runtime (runs fast on all computers)
6. Images

[ --> -->Easy To use<-- <-- ]

7. Linear, and Radial Built into XML
8. Opacity!
9. Rectangle, and Ellipse shapes

///////////////////////////////////////////////////////////////////////

View an Example (after you have downloaded the runtime)

(the media player itself, is SilverLight.)

Requirements: (1.0 beta)
1. You need to know a little xml
2. You need to know Javascript.
3. You need to know Html.

Requirements: (1.1 alpha)
1. You need to know a little xml
2. You need to know javascript.
3. You need to know C#.
4. You need to know a lil Asp.net
