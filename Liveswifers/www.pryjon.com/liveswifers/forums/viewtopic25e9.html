OVERVIEW: This tutorial is meant to help you program gravitational pull for a certain object. Note that this is just for the gravitational pull, and that this tutorial does not show you how to slow it down by tapping the up key (like those gravity ball games). This is just the gravity pull part.
NOTICE: I cannot test this code due to technical problems. If you find any errors, please feel free to correct me.
-----------------------
OK, this is simple, so here is what you do:
Create the sprite that you want to put gravitational pull on. Then drag the sprite from the library to the main canvas.
In the sprite's onLoad event, put this code:
Code:
var accelSpeed=0;

In the sprite's onEnterFrame event, put this code:
Code:
this._y=this._y + 1;

Change 1 to a higher number for faster pull. And there you have the simplest gravitational pull code. But read on for more!
-----------------------
Now, let us build on the sprite's onEnterFrame code so that it accelerates:
Code:
accelSpeed=accelSpeed+1
this._y=this._y+accelSpeed;

There you go! Your basic acceleration code!
------------------------
Now, to make it stop when the sprite hits a platform, you first make a platform in a sprite. Drag the platform onto the main canvas and give it the instance name "platform". Then, the give the falling sprite an instance name of "player". Now, enter this in the sprite's (with the instance name "player") onEnterFrame event:
Code:
if (_root.player.hitTest(_root.platform)) {
    accelSpeed=accelSpeed+1
    this._y=this._y+accelSpeed;
}
else {
    accelSpeed=0;
}

Hope this helps!
Note: From experience, this code:
Code:
if (_root.player.hitTest(_root.platform))

Is the only way you can make this conditional work.
