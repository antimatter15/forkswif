I put a tutorial in that section. I wish I could find an easier way to do it though. The way I accomplished it was to create a semi-transparent sprite called cam with the instance name "cam" with
Quote:
_visible = false;
in the sprite's actionscript code to make it disappear when you export the swf and then in the maintimeline I added another layer which had only
Quote:
_root._x = -_root.cam._x*(500 /_root.cam._width);
_root._y = -_root.cam._y*(400 /_root.cam._height);
_root._xscale = 100 *(500 /_root.cam._width);
_root._yscale = 100 *(400 /_root.cam._height);
in the actionscript code.

You would have to replace 500 with your width, and 400 with your height. and then you would just want to make sure that this code, and the other layer with the "camera" are on each frame of your movie.

So, really you would just add these two extra layers to your movie. Make sure the camera is on top, and you can resize and move it around, either frame by frame, or tween the movement. Whatever is contained by the camera's box is what will show up on the screen. The actionscript that is copied in the main timeline just changes the stage scale, width and height to the camera scale, width and height, more or less.

I based it off of the actionscript camera for flash, but I adjusted it for the (x,y) differences in liveswif, and got rid of the code I couldn't make work. I am trying to see if I can get some code to work just in a sprite, so that way you would be able to just put the sprite instance on stage and not have to worry about copying actionscript across the length of the movie. I guess we'll see if I can get it to work.

EDIT - I did get it to work. I'll update the tutorial. If you create another sprite, say, a small circle. In the actionscript when you create the sprite, put in the
Quote:
_root._x = -_root.cam._x*(500 /_root.cam._width);
_root._y = -_root.cam._y*(400 /_root.cam._height);
_root._xscale = 100 *(500 /_root.cam._width);
_root._yscale = 100 *(400 /_root.cam._height);
code in both the first and second frames. You only have to have your small object in the first frame, then you won't have to copy actionscript across the whole darn movie, you can just insert it on it's own frame, keep it off stage so you will never have to look at it, and then just transform tween it so it will last the length of the movie. Remember to change the code for the height and width.
