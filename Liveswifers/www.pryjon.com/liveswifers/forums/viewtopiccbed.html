The Ajax Animator will be *completely* rewritten. in the upcomming version 0.20, where we skip straight from 0.15-0.20 (the biggest leap the app has ever gone through)

There will be ABSOLUTELY NO code that is "ported", every line will be rewritten.

I have to say, the project is INCREDABLY promising. With built-in animation-management, better community aspects, a completely new file format, IE support, a better revisions and clipboard management, a community-maintained clipart library, plugins, 16+ themes, long-abandoned features like the manual and FAQ, more icons, bugfixes, new images/icons, almost compeltely different UI, more manageable timeline, usability features, better tweening, tweening opacity, line width, line color, fill color, etc and tweening via quadratic paths.

It also switches to GPL v3, Ext 2.1, OnlyPaths, a new color picker. and possibly from the PHP backend to a Python one.

If you want a taste of the new drawing editor:
http://aamap.110mb.com/animator/onlypaths/

and the actual application (no integration between drawing editor yet...)
http://antimatter15.110mb.com/animator/Animator2/build/ajaxanimator.htm

The thing is, that this version has a new update/compile management system. so I can LITERALLY press 2 buttons and compile/upload the new versions to my servers. Then I can make updates following the google code "Release Early, Release Often" slogan. without all those Subversion troubles.
