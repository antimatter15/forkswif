Well the following isn't much of a tutorial, but here's a few tips on how to improve your LiveSwif movies.

1. For each part of your characters, use a different layer, unless you are using "action sprites". This means that every part of every character gets its own layer, but there are exceptions. If you're not going to animate certain parts, you can put them on the same layer.

2. Use Actionscript on buttons so that they can be used as play/replay buttons. To do this, simply click the button which will contain the Actionscript and type in the ActionScript box:

Code:
stop();
gotoAndPlay(*frame number goes here*);


... where *frame number goes here* is the frame which your actual movie starts on.

The gotoAndPlay code can also be used if you want to keep the user at a part of your movie until they click a button. Just go to the frame just after where the part you want ends and enter the code as above, but leave out the stop; and you don't need a button.

3. If you want to use a complex animation but you don't want to animate it every time you need it, simply make an "Action Sprite".

Here is a tutorial to do so.

4. Sometimes you'll find that you can't make the designs you want just using the Super shapes function. Just use the Curve Tool to manipulate the shapes.
Also, you may also find that no matter how hard you try, you can't seem to get rid of the lines you don't want. A simple solution is to make a smaller shape using the Supershape tool, getting rid of the lines by going to the Select Tool and putting 0 in the Line box (Don't forget to click Apply!) and making it the same colour as the area surrounding the line you want rid of. Then, shape and movie it to fit by either using the Curve tool, or rotating and resizing it so that it covers the line. You can use this on the same layer as the part you want to cover up.

Provided you did put them on the same layer, you can group the shapes together, if you're not going to animate the part much.

5. Always, always, ALWAYS check your Flash. Check it using the "Preview" button in LiveSwif, and check it when you save it!
That way, you can fix errors before you put the movie on the Internet.

6. Use the tutorials and the forum for help. They're there for a reason!

7. Draw your characters facing 3/4 of the side they are supposed to be facing (or as close as you can get to it ! ), instead of facing just the front or just the side.
It should allow more features to be visable, allowing more expression for your characters, improving your Flash.

Well, that's all I can think of. Hope that helps!
