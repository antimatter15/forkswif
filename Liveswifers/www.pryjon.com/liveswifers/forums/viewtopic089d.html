Here is one for PHP:
Code:
function hex2rgb($hex){
   str_replace('#', '', $hex);
   $red=-1;
   $green=-1;
   $blue=-1;
   if(strlen($hex)==3){
      $red=hexdec($hex[0].$hex[0]);
      $green=hexdec($hex[1].$hex[1]);
      $blue=hexdec($hex[2].$hex[2]);
   } else if(strlen($hex)==6){
      $red=hexdec($hex[0].$hex[1]);
      $green=hexdec($hex[2].$hex[3]);
      $blue=hexdec($hex[4].$hex[5]);
   }
   return array('red'=>$red, 'green'=>$green, 'blue'=>$blue);
}


It takes in a hex string, like 'ffee44', and returns an array of 3 decimals like this: Array ( [red] => 255 [green] => 238 [blue] => 68 );. It does absolutely no error checking though, other than making sure the incoming $hex string is either 3 or 6 characters.
