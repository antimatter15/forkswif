Uhh, it's kinda hard to explain how to program a view, but I'll try my best to explain.

First, use this tutorial to make a virtual camera.

When you're done with that, make two identical rectangle sprites that have the same height as the camera box but maybe a fourth of the width (keep in mind that although they're identical, they still have to be seperate sprites| you can make them different colors if you want, just make sure they're the same size).

Next, edit the camera sprite by double clicking on it and once you're in it, drag the two sprites that you previously made from the library to the editing screen. Put one on the far left side of the camera box and one on the far right side of the camera box and line them up vertically with the camera box.

Click on the one on the right side and name the instance name rightco, click the one on the left side and name the instance name leftco.
When your finished with that it should look something like this:


Now go back to the main scene or whatever it's called (unless you changed it, it would be Scene 1).

Click on the cam sprite and in the onClipEvent(enterFrame) section use this code:
Code:

if(this.rightco.hitTest(_root.player)){
    _x+=PICKANUMBER;
}
if(this.leftco.hitTest(_root.player)){
    _x-=PICKANUMBER;
}

Replace the PICKANUMBER with whatever speed you're player moves at. And replace _root.player with watever you're player's instance name is.

Hopefully, if all goes well, it'll work.
