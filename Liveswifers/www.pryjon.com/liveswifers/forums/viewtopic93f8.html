To be blunt, forget trying to reverse engineer the current LiveSwif. It ain't gonna happen.

If you can't get ahold of the *real* source code to LiveSwif, I suggest trying to find a codebase you *can* work from.

A couple possiblities:
UIRA - Open source, available for Win & Lin, but a very young project
Powerbullet Presenter - Abandonware like LiveSwif but not as powerful, but in this case at least the author has said he's considering releasing the source code.

Might be worth getting in touch with some of the guys over on Flashcooker too, and do some searching on SourceForge for some flash-related code you could use.
