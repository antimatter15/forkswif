Ok here is the solution to that pain in the ass bug.

It's really quite simple. use the freehand tool and draw a line beside your shape (this should be the shape that shows the horizontal lines during preview).

Then select both shapes & use the "add shape" function.

Now select your new shape & use the "break shape" function.

Delete the line you drew beside the shape.

Voila! You have made your way around the bug.

Note: Recolor your shape if necessary.

ok, let me re-phrase everything

1 ) First, find the culprit of the lines.

2a ) Then, select the shape.


2b ) Go to the tool dropdown menu.

2c ) Check if the option "ungroup" can be clicked, if it is, click it.

2d ) Check if the option "Convert to Shape" can be clicked, if it is, click it.

3 ) Next find the shape that is creating the white line, and using the freehand tool, draw a small straight line.

4 ) Select both the line you made, and the shape with the white line.

5 ) With those items selected, go to Tool>Combine>Add shapes

6 ) The problem has been fixed, although you may lose the color, the shape will be a little bit distorted, and the line and the shape has been merged together.

7 ) Now select the shape with the line and the culprit of the white line.

8 ) With the shape selected, go to Tool>Break shape.

9a ) Now then, select the line only and delete it.

9b ) Select the new shape and re-color it.

And there you go!
