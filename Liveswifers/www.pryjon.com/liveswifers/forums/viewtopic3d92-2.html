Hi all,

Not exactly a Flash IDE replacement, I've been writing a Captivate improvement/replacement (etc) IDE for the last two years or so. It's written from scratch, under the LGPL license, so it's all properly Open Source. And it works on both Windows/Linux/Solaris/etc. (haven't tried Mac OS X though, so I dunno)

http://www.flameproject.org

I was more focused on getting animated SVG happening, but the announcement of no animated SVG in Firefox 3 was a bit of a motivation killer for that direction.

So... I've started adding in swf support. Been coding that (from scratch) for a couple of weeks now, and will probably have something usable in about a month at the present rate. (And no, completely not interested in using ming in its present state, have you seen the docs for it?)

I'm ok with coding the swf byte code back end and similar, but suck at graphics programming. i.e. making things like collision detection work well, drag and drop objects in the working area onscreen, etc.

Anyone interested in contributing time/effort towards helping?

Regards and best wishes,
